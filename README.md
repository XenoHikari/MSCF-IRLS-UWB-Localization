# MSCF-IRLS-UWB-Localization

This repository provides the MATLAB simulation code for the proposed MSCF-IRLS method in UWB TDoA-based localization.

The code is associated with the paper:

**"[MSCF-IRLS: Multi-Subset Consistency Fusion and IRLS Refinement for UWB Localization with Concentrated Anchors]"**

The purpose of this repository is to improve the reproducibility and transparency of the simulation results reported in the paper.

---

## Description

The proposed MSCF-IRLS method is designed for robust UWB TDoA localization under measurement noise and non-line-of-sight errors.

This repository includes MATLAB scripts for:

- Running the proposed MSCF-IRLS algorithm
- Comparing the proposed method with baseline algorithms
- Reproducing the main simulation results
- Generating figures and tables used in the paper

---

## Requirements

The code was developed and tested in MATLAB.

Recommended environment:

- MATLAB R2021a or later
- Windows 10/11

Please add all folders and subfolders to the MATLAB path before running the code.
